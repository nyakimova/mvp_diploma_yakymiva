Цей репозиторій містить програмну реалізацію мінімально працюючої версії системи (MVP) для оцінки фінансового ризику надання медичних послуг на основі даних про витрати, тарифи, обсяг послуг та операційні фактори.

Система розроблена в межах роботи над проєктом та поєднує:

синтетичну генерацію великого датасету,

машинне навчання (класифікація рівнів ризику),

вебінтерфейс для введення даних та отримання прогнозу.
